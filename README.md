# Exercise Reminder App

## 🎯 Wertversprechen

Wir helfen Menschen mit sitzenden Berufen (im Home-Office) und Rückenschmerzen dabei, ihre Rückengesundheit zu fördern, indem wir sie daran erinnern, Pausen einzulegen und körperliche Übungen durchzuführen.

## ✨ Funktionen

- Automatische Erinnerungen für regelmäßige Pausen
- Gezielte Rückenübungen
- Benachrichtigungen für Übungszeiten

## 🚀 Projekt starten

### Voraussetzungen

- Node.js & npm installiert
- Expo CLI: `npm install -g expo-cli`

### Installation & Start

```bash
npm install
npx expo start
```

### App ausführen

- **Simulator (iOS):** Press `i` zum iOS Simulator zu öffnen
- **Android Emulator:** Press `a`
- **Handy:** Expo Go App scannen QR-Code

### ⚠️ Bekannte Einschränkungen

- Notifications funktionieren nur auf dem physischen Gerät (nicht im Simulator)
- iOS Simulator benötigt Xcode
